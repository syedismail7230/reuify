API-Application Programming Interface

API allows apps to send information between each other. While there are numerous protocols and technologies involved, the underlying purpose of APIs is always the same: to let one piece of software communicate with another. Application Programming interface is an abbreviation that means "application programming point of interaction," and it permits applications to send data between one another. While there are various conventions and advances included, the basic motivation behind APIs is similar all the time: to let one piece of programming speak with another. RestAPI

The REST API (also called the RESTful API) is an application programming interface (API or web API) that conforms to the limitations of the REST architecture and allows interaction with RESTful web services. REST stands for Representative State Transfer and was created by computer scientist Roy Fielding.

APIGEE

Apigee is an API development and management platform. By dealing with proxy layer services, Apigee provides an abstraction or facade for your backend API services and provides security, speed limits, quotas, analytics and more.

Django

Django is a high-level Python web framework that enables the rapid development of secure and sustainable websites. Created by experienced developers, Django takes care of most web development issues, so you can focus on writing your app without having to change the wheel.
